# ChatGPT Messenger Clone

ChatGPT Messenger project inspired by Sonny Sangha from: https://www.youtube.com/watch?v=V6Hq_EX2LLM

## Tech Stack

- Next.js 13 (Experimental app dir feature)
- Tailwind CSS
- Firebase Firestore
- Firebase Auth
- NextAuth.js
- OpenAI API

## Environment variables

```
GOOGLE_ID=
GOOGLE_SECRET=
NEXTAUTH_URL=
NEXTAUTH_SECRET=
FIREBASE_SECRET=
OPENAI_SECRET=
```
